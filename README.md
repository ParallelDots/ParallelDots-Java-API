# ParallelDots-Java-API
Java wrapper for ParallelDots APIs

# Installation
#### Use the JAR
* paralleldots-1.0-SNAPSHOT.jar

#### Path to JAR
> path: target/paralleldots-1.0-SNAPSHOT.jar

#### Dependencies
* okhttp-3.10.0.jar
* okio-1.14.0.jar
* json-simple-1.1.jar

#### Languages Supported
	- Portuguese ( pt )
	- Simplified Chinese ( zh )
	- Spanish ( es )
	- German ( de )
	- French ( fr )
	- Dutch ( nl )
	- Italian (it)
	- Japanese ( ja )
	- Russian ( ru )
	- Thai ( th )
	- Danish ( da )
	- Finish ( fi )
	- Arabic (ar)
	- Greek (el)
